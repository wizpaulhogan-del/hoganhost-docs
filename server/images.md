# Available Images

You can specify the image slug when creating a server.  
Some examples include:

- **Ubuntu**
  - ubuntu-22-04-x64
  - ubuntu-20-04-x64
  - ubuntu-18-04-x64

- **Debian**
  - debian-11-x64
  - debian-10-x64

- **Fedora**
  - fedora-38-x64
  - fedora-37-x64

- **CentOS / Rocky / Alma**
  - centos-7-x64
  - rocky-8-x64
  - almalinux-8-x64
  - almalinux-9-x64

- **Marketplace (Apps)**
  - docker-20-04
  - wordpress-20-04
  - lamp-20-04
  - lemp-20-04
